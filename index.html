Procurando um desenvolvedor FullStack?

Julio Cesar Cioato Vanz Foco, disciplina e consistência. Vamos revolucionar o mercado de tecnologia juntos?
Olá, meu nome é Julio, tenho 19 anos e atualmente estou procurando um emprego/estágio na área de desenvolvedor/programador. No momento estou aprendendo a usar as linguagens de JavaScript, CSS, HTML, entre outras através do curso Profissão Programador, onde já aprendi a desenvolver este portfólio, usando as linguagens acima citadas, entre outras coisas. Estou aprimorando meus conhecimentos desenvolvendo um site, onde simula uma página de lançamento de um filme, tendo um blog sobre as curiosidades do filme e uma página com comentários, ainda, exibindo banners dos autores principais, bem como, suas fotos, o desenvolvimento do site é bem básico, é apenas para aprimorar meus conhecimentos de HTML e CSS. Pretendo ingressar no mercado de trabalho o mais rápido, aderindo o máximo de conhecimento possível, para progredir cada vez mais.

Nesse projeto foi usado as ferramentas HTML e CSS. Nele foi criado a fim de melhorar e aprimorar os meus conhecimentos. Esse projeto se baseia em uma página do novo filme do Batman, possuindo as fotos dos três princiais atores do filme e possuindo uma página com curiosidades do filme, assim como o trailer do filme. Contem também um botão com a opção de comprar ingressos.

Nesse projeto usei as ferramentas HTML, CSS e JavaScript. Esse projeto serve para calcular seu indice de massa corporal, ao calcular usando sua altura mais seu peso, ele vai te dar a resposta se você está abaixo do peso, no peso ideal, com obesidade nivel um, dois ou três. Projeto pequeno a fim de melhorar as habilidades com o JavaScript.
